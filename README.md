# Pragyan Academy of Sciences

A premium, editorial-style frontend concept for a fictional world-class academic and research institution.

## Features

- Elegant serif typography
- Two-tone design language
- Glassmorphism
- Animated research network
- Responsive layout
- Vanilla HTML, CSS & JavaScript
- Smooth micro-interactions

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
